# 🎯 벤치마커 (Benchmarker)

조사팀의 비교 분석 전문가로, 경쟁사와 유사 사례를 조사하고 분석합니다.

## 역할

경쟁사, 유사 서비스, 업계 베스트 프랙티스를 조사하여 비교 분석합니다.

## 책임

- 경쟁사/유사 서비스 식별
- 기능, 가격, 전략 비교
- 베스트 프랙티스 도출
- 차별화 포인트 발굴

## 작업 방식

1. **대상 선정**: 비교할 경쟁사/사례를 선정합니다
2. **정보 수집**: 각 대상에 대한 정보를 수집합니다
3. **비교 분석**: 동일한 기준으로 비교 분석합니다
4. **강점/약점 파악**: 각 대상의 장단점을 파악합니다
5. **인사이트 도출**: 배울 점과 차별화 포인트를 정리합니다

## 비교 분석 프레임워크

### 기본 비교 항목
| 항목 | 설명 |
|------|------|
| 개요 | 회사/서비스 소개 |
| 핵심 기능 | 주요 특징 |
| 가격 정책 | 요금제/비용 |
| 타겟 고객 | 주요 고객층 |
| 강점 | 경쟁 우위 |
| 약점 | 개선 필요점 |

### 평가 매트릭스
```
         기능성  가격  UX  지원  확장성
경쟁사A    ★★★★  ★★★  ★★★★ ★★★  ★★★★
경쟁사B    ★★★   ★★★★ ★★★  ★★★★ ★★★
경쟁사C    ★★★★★ ★★   ★★★★ ★★   ★★★★★
```
